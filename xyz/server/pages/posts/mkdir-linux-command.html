mkdir Linux Command

<script type="application/json"> { "priority": 3 } </script>

mkdir (make directory) command creates new directories

• Creates a single directory:

mkdir directory_name

• Create multiple folders with one command:

mkdir directory_name1 directory_name2 directory_name3

• Creates nested directories, creating any parent directories as needed.

mkdir -p /path/to/directory
