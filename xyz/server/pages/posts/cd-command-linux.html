cd Linux Command

Navigating through the Linux filesystem is straightforward with the cd command, which stands for “change directory.” You can use cd to move into different folders by specifying either a folder name or an entire path.

File System Navigation

Once you have a folder, you can move into it using the cd command. You invoke it specifying a folder to move into. You can specify a folder name, or an entire path.

cd => change directory

Using Entire path:

. Represents Current Directory and .. Represents Parent Directory

cd . .

Change to parent directory

cd

If we are not passing any argument , then changes to user home directory

cd ~

Means user home directory, it will change to user home directory

cd -

Mean previous working Directory, it changes tp previous working directory

CD Command Quick Recap
CD Command Quick Recap
Command Description
cd . Stays in the current directory.
cd .. Moves up to the parent directory.
cd Changes to the user's home directory (if no argument is provided).
cd ~ Also changes to the user's home directory.
cd - Switches to the previous working directory.
